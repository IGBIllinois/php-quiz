# PHP-Quiz Manual

## Logon
* Login with your username and LDAP credentials


## User View
* Select the quiz you would like the take and click on Take Quiz
* Answer each question
** Passed: you will see a green congratulations banner box with an option to print a certificate
** Failed: you will see a red banner with Failed: Quiz was reset, please try again.

## Admin View
* Cick on the Admin link on the upper left

## Edit Quiz
* In edit quiz you can either add/edit/delete a quiz.
* When editing a quiz you have the option to edit individual questions or delete/deactivate questions.
* When you delete a quiz or a question it simply deactivates, you can reactivate it later if you wish.

## User Quiz Results
* Choose a quiz and click on View Results next to it to view user results
* Click on the percentage next to the user you would like to view results for to view answered questions
* You can view what a user answered.

##User Permissions
* Select a user form the list and set them as admin or moderator.
** Moderator: Can only view quiz results
** Admin: can edit quizzes and change permissions.
